# Configuration Prospectius

## Paramètres MySQL par défaut

```
Host: localhost
Port: 3306
User: root
Password: root
Database: Prospectius
```

## Modifie ces paramètres dans l'écran de configuration au premier lancement

Si tu utilises MariaDB ou un serveur MySQL externe, tu peux modifier:
- L'adresse IP ou le nom du serveur
- Le port (si différent de 3306)
- L'identifiant utilisateur
- Le mot de passe

Ces paramètres seront sauvegardés après la première connexion réussie.
